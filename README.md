# Kuni Hub Scripts

A clean, smooth, and curated collection of Roblox scripts, organized by game and regularly updated.

This site provides easy access to working scripts with preview images, copy-ready loaders, and a responsive interface for both PC and mobile users.

## Features

- Clean white & gold UI
- Script preview images
- One-click copy loader
- Game-based filtering (Adopt Me, MM2, Others)
- Fast and mobile-friendly
- Regularly maintained scripts

## Supported Games

- Adopt Me
- Murder Mystery 2
- Other popular Roblox games

## Usage

1. Open the website via GitHub Pages  
2. Browse or search for a script  
3. Click **Copy** to copy the loader  
4. Paste the loader into your executor  
5. Execute in-game

## Notes

- Scripts are provided as-is
- Status shows whether a script is currently working
- Some scripts may stop working after game updates
